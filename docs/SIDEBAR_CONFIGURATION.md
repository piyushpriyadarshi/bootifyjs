# Sidebar Configuration Guide

## Current Status

The sidebar configuration in `sidebars.ts` is currently set to **autogenerated mode** to work with the existing tutorial content. This is a temporary configuration.

## Complete Navigation Structure

The complete sidebar structure for BootifyJS documentation has been defined in `sidebars.ts` but is currently commented out. It includes three main sidebars:

1. **docsSidebar** - Main documentation with sections:

   - Getting Started
   - Core Concepts
   - Modules (Core, Events, Cache, Logging, Auth, Config)
   - Advanced Topics
   - Guides
   - Contributing

2. **apiSidebar** - API Reference documentation:

   - Decorators
   - Core API
   - Events API
   - Cache API
   - Logging API
   - Auth API
   - Config API

3. **templatesSidebar** - Code templates:
   - REST API
   - Authentication
   - CRUD Operations
   - Event-Driven
   - Caching Patterns
   - Error Handling
   - Middleware
   - Testing
   - Database Integration
   - Full Application

## Activating the Complete Structure

Once the documentation files are created (tasks 3-21), follow these steps:

1. Open `docs/sidebars.ts`
2. Remove the autogenerated configuration:
   ```typescript
   docsSidebar: [{type: 'autogenerated', dirName: '.'}],
   apiSidebar: [{type: 'autogenerated', dirName: '.'}],
   templatesSidebar: [{type: 'autogenerated', dirName: '.'}],
   ```
3. Uncomment the complete sidebar structure (marked with `/* ... */`)
4. Test the build with `npm run build`
